## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
**File**: `crates/extension/src/data.rs:184-189`

**What**: `register()` formats `provider_name = %name` and `dropped_provider_reports_name = %provider.name()` through Display. While `name()` returns &'static str, the `name` parameter is `impl Into<String>` and could be runtime-generated by an extension reading external data. A name with `\\n` or ANSI sequences forges log entries.

**Why it matters**: TASK-1127 covers `log_and_redact_spawn_error` for `spec.program`; this is a parallel hit on the data-registry diagnostic path with identical class.
